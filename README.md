# Приложение Kanban доски

React-приложение для управления задачами в формате Kanban доски, построенное с использованием Vite. Включает управление задачами, аутентификацию пользователей и поддержку светлой/темной темы.

## Возможности

- **Управление задачами**: Создание, редактирование, удаление и организация задач в формате Kanban доски
- **Аутентификация пользователей**: Функционал входа и регистрации
- **Поддержка тем**: Переключение между светлой и темной темой
- **Адаптивный дизайн**: Работает на десктопе и мобильных устройствах
- **Валидация форм**: Проверка ввода для предотвращения пустых или состоящих только из пробелов полей
- **Продвинутые уведомления**: Система всплывающих уведомлений для обратной связи с пользователем
- **Drag & Drop**: Перетаскивание задач между колонками для быстрого изменения статуса
- **Обработка ошибок**: Грамотная обработка ошибок сервера с fallback на локальное хранилище
- **Анимации загрузки**: Улучшенные индикаторы загрузки со спиннерами

## Реализация темной темы

Приложение включает комплексную систему темной темы, реализованную с использованием styled-components и React Context.

### Ключевые компоненты:

- **ThemeProvider**: Оборачивает все приложение и предоставляет контекст темы
- **Theme Context**: Управляет состоянием темы и функцией переключения
- **Styled Components**: Все UI компоненты используют стили, зависящие от темы
- **Сохранение темы**: Выбранная тема сохраняется в localStorage

### Использование:

1. Нажмите на меню пользователя в заголовке
2. Переключите чекбокс "Темная тема" / "Светлая тема"
3. Тема меняется мгновенно и сохраняется между сессиями

### Технические детали:

- Темы определены в `src/themes.js` с палитрами цветов для светлого и темного режимов
- `ThemeContext` использует `styled-components` ThemeProvider для глобального доступа к теме
- Компоненты получают доступ к цветам темы через `props.theme.colors`
- Логотип меняется динамически в зависимости от выбранной темы

## Реализация продвинутых уведомлений

Приложение включает систему продвинутых всплывающих уведомлений, реализованную с использованием библиотеки React-Toastify.

### Ключевые компоненты:

- **ToastContainer**: Глобальный контейнер для отображения уведомлений
- **Утилиты уведомлений**: Функции `showSuccess`, `showError`, `showInfo`, `showWarning` в `src/utils/toast.js`
- **Интеграция в компоненты**: Уведомления используются во всех формах валидации и операциях CRUD

### Использование:

Уведомления автоматически появляются при:
- Успешном входе/регистрации
- Ошибках валидации форм
- Успешном создании, редактировании или удалении задач
- Ошибках API запросов

### Технические детали:

- Уведомления настраиваются через `ToastContainer` в корневом компоненте `App.jsx`
- Утилиты импортируют `toast` из `react-toastify` для показа уведомлений разных типов
- Заменяют традиционные `alert()` и текстовые сообщения об ошибках в формах
- Поддерживают авто-закрытие и пользовательское закрытие

## Реализация Drag & Drop

Приложение включает функциональность перетаскивания задач между колонками, реализованную с использованием библиотеки @dnd-kit.

### Ключевые компоненты:

- **DndContext**: Глобальный контекст для управления перетаскиванием
- **SortableContext**: Контекст для сортируемых элементов в колонках
- **useSortable**: Хук для создания перетаскиваемых элементов
- **useDroppable**: Хук для создания зон сброса (колонок)
- **DragOverlay**: Визуальный оверлей во время перетаскивания

### Использование:

1. Захватите задачу за заголовок (название задачи)
2. Перетащите в нужную колонку статуса
3. Отпустите для применения изменений
4. Статус задачи обновится автоматически

### Технические детали:

- Зона перетаскивания ограничена только заголовком задачи для предотвращения конфликтов с кнопками
- Колонки выступают в качестве droppable зон с уникальными идентификаторами
- Визуальная обратная связь включает прозрачность перетаскиваемого элемента
- Интеграция с TaskContext для сохранения изменений на сервере и в localStorage
- Поддержка клавиатурной навигации через KeyboardSensor

## Начало работы

### Предварительные требования

- Node.js (v14 или выше)
- npm или yarn

### Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
4. Откройте [http://localhost:5173](http://localhost:5173) в браузере

```
│   ├── PopNewCard/     # Модальное окно новой задачи
│   ├── PopBrowse/      # Модальное окно деталей задачи
│   └── ...
├── context/            # React контексты
│   ├── ThemeContext.jsx # Управление темой
│   ├── AuthContext.jsx  # Аутентификация
│   └── TaskContext.jsx  # Управление состоянием задач

├── services/           # Сервисы API
├── utils/              # Утилиты и хелперы

```

## Используемые технологии

- **React**: Библиотека UI
- **Vite**: Инструмент сборки и dev сервер
- **styled-components**: CSS-in-JS стилизация с поддержкой тем
- **Axios**: HTTP клиент для API запросов
- **React Router**: Клиентская маршрутизация
- **React-Toastify**: Библиотека для продвинутых всплывающих уведомлений
- **@dnd-kit**: Библиотека для реализации Drag & Drop функциональности
